# データ整形：ここが9割

* ルール：

  * **1行目は見出し**（項目名）
  * A列に**カテゴリ/時系列**、2列目以降が**系列**
  * **結合セル・改行**は避ける、空白は空欄 or 0 を明確に
* 表は**長すぎてもOK**。グラフは後で**範囲選択**
* 数値書式（桁区切り・小数点）を**先に整える**と誤解が減る

> グラフが崩れる時は、まず**表の形**と**空欄の扱い**を疑う。

---

# どのグラフを使う？（使い分け）

|     目的     |      推奨グラフ       |               補足                |
| ------------ | --------------------- | --------------------------------- |
| 値の比較     | 縦棒（集合/積み上げ） | 0起点が基本（棒の長さで比較）     |
| 時系列の傾向 | 折れ線                | マーカー最小限、月/日目盛を適切に |
| 相関（関係） | 散布図 + 近似直線     | R²の表示で説明力を示せる          |
| 割合構成     | 100%積み上げ棒        | 多カテゴリの**円**は避ける        |
| 分布         | ヒストグラム          | ビン幅を明記（今回は紹介のみ）    |

> **3Dグラフは原則禁止**。情報より見た目が勝ち、誤読の原因。

---

# 実演①：縦棒グラフ（カテゴリ比較）

1. 範囲選択：`カテゴリ ~ 系列`
2. **挿入>縦棒**
3. 仕上げ：

   * **軸の最小値=0**（棒）
   * **データラベル**（必要最小限）
   * 凡例の位置を**上** or **右**へ
   * 罫線/グリッドは**薄く**

> 見出しは**内容が伝わる文**に（例：*学科別レポート提出数（2025後期）*）。

---

# 実演②：折れ線（時系列）+移動平均

* 例：`日付`×`PV`
* グラフ：**折れ線**
* 移動平均（7日）列を追加：
  `=AVERAGE(C2:C8)` を**下へ**。系列として追加
* 目盛間隔を月/週で調整、**マーカーは控えめ**

> 迷ったら、**線1本＋薄いグリッド**の極力シンプルな画面に。

---

# 実演③：散布図 + 近似直線 + R²

1. `X列, Y列` を選択 → **挿入>散布図**
2. **近似直線（線形）**を追加
3. **式とR²をグラフに表示**（相関の説明）
4. 軸の単位・範囲を**実データに合わせる**（外れ値に引っ張られない）

> 棒グラフに**第2軸**を乱用しない。相関は**散布図**一択。

---

# グラフの仕上げ5点セット

1. **軸ラベル**：量の名前＋**単位**（例：温度[°C]）
2. **桁・書式**：小数点/桁区切り/百分率
3. **色**：系列の区別だけ。**意味のない虹色は避ける**
4. **凡例**：重ならない位置。1系列なら**削除**も可
5. **注釈**：必要箇所に**矢印**や**テキストボックス**

> 迷ったら**白・灰・強調色1色**。色弱者配慮の濃淡設計が安全。

---

# 図形で注釈（伝わる説明）

* **矢印／吹き出し**でピーク・外れ値・変化点を指示
* **グループ化**して位置ズレを防止
* **整列/間隔**（右クリック>配置）で揃える
* **Alt**キーでセルにスナップ（Windows）

> 注釈は“最小限・端的”。説明文は**本文側**で詳述。

---

# 論文におけるグラフ（原則）

* **図番号**と**キャプション**：図の下に *図1 実験Aの温度推移（平均±SD）*
* **本文内参照**：*図1に示すように…*
* **再現性**：

  * データの出所、前処理（例：移動平均7日、外れ値除外規則）
  * 軸範囲・ビン幅・近似の種類
* **誤読を避ける**：0起点（棒）／縦軸倍率の明示／割合は100%基準
* **原図の解像度**：印刷時でも**文字が読める**大きさ

---

# チェックリスト（提出前）

* [ ] 図タイトルと**軸ラベル・単位**がある
* [ ] 0起点（棒）／線は適切な範囲・間隔
* [ ] 凡例は重ならない・不要なら削除
* [ ] 文字サイズは本文相当（9–11pt目安）
* [ ] 注釈は簡潔、色は控えめ（強調1色）
* [ ] キャプションに**条件・処理**が書かれている

## 12) よくあるNG

* 3D円グラフ／極端なグラデーション
* 0起点でない棒、単位の欠落
* 凡例がデータに**重なる**、文字が小さすぎる
* 色が多すぎて**意味がない**

---

## 参考（小ワザ）

* **Ctrl+`**：式の表示切替（移動平均の確認）
* **右クリック>データの選択**：系列の入替
* **線の太さ**：1.0–1.5pt程度が見やすい
* **エラーバー**：平均±SD/SE（必要時のみ）

---

## まとめ

* グラフは**データの主張を支える証拠**。装飾より**可読性**
* 図番号・キャプション・本文参照で**論文の流儀**に合わせる
* 今日の3種（棒・線・散布）を**丁寧に**仕上げられれば合格
